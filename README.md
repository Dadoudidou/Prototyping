# Prototyping

## Commencement
```
npm install
```

`npm run start` lance un serveur de développement.


`npm run publish` compile les fichiers

## Routes
Créer un fichier `index.tsx` dans un répertoire du dossier `src/routes/`; L'url correspondera au chemin du fichiers.
```
+ src
+-- routes
+---- A
+------ B
+-------- index.tsx

L'url sera /A/B
```
